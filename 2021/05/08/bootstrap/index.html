一.布局容器

bootstrap提供了两种布局容器类

  • .container 类用于固定宽度并支持响应式布局的容器。
  • .container-fluid 类用于100%宽度，占据全部视口的容器。

1、.container 类用于固定宽度并支持响应式布局的容器

// 固定宽度，会有留白

<div class="container">
    ...
</div>

2、.container-fluid 类用于100% 宽度，占据全部视口（viewport）的容器

<div class="container-fluid">
    ...
</div>

二.栅格布局

BootStrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。栅格系统用于通过一系列的行（row）或列（column）的组合来创建页面布局，你的内容就可以放入这些创建好的布局中。

网格系统的实现原理非常简单，仅仅是通过定义容器大小，平分12份（也有平分成24份或32份，但12份是最常见的），再调整内外边距，最后结合媒体查询，就制作出来强大的响应式网格系统。BootStrapu框架中的网格系统就是将容器平分成12份。

注意：网格系统必须使用到CSS

1. 列组合：

container、row、xs（xsmall phones），sm（small tablets），md（middle desktops），lg（laege desktops），即

超小屏（自动），小屏（750px），中屏（970px）和大屏（1170px）

自动会媒体查询，所以一个元素中可以添加多个：

<div class="col-md-4 col-lg-8">4列</div>

数据行（.row）必须包含在容器（.container）中，以便为其赋予合适的对齐方式和内距（padding）。

在行（.row）中可以添加列（.column），只有列（column）才可以作为行容器（.row）的直接子元素，但列数之和不能超过平分的总列数，比如12。==如果大于12，则自动换到下一行。==

具体内容应当放置在列容器之内。

<div class="container">
    <div class="row">
        <div class="col-md-4">4列</div>
        <div class="col-md-8">8列</div>
    </div>
</div>

2. 列偏移

如果不希望相邻的两个列紧靠在一起，可以使用列偏移(offset)功能来实现。

只要在列元素上添加类名 col-md-offset-*（星号表示要偏移的列组合数）具有这个类名的列就会向右偏移。

<div class="col-md-2 col-md-offset-2">
                向右偏移二个
</div>

3. 列排序

列排序就是改变列的方向，改变左右浮动，并且设置浮动的距离，通过添加类名：

  • col-md-pull-* 向前浮动
  • col-md-push-* 向后浮动 （星号代表移动的列组合数）
<div class="row">
            <div class="col-md-2 ">
                浮动起来
            </div>
            <div class="col-md-2">
                原本的元素
			</div>
</div>
  • 列偏移时，如果被偏移的列后还有元素，后面的元素会跟着一起偏移。
  • 列排序时，如果被浮动的列后还有元素，==后面的元素会覆盖前面的元素。== (覆盖关系和谁浮动无关，只和位置有关)

4. 列嵌套

Bootstrapz框架的网格系统可以支持列的嵌套，可以在一个列中添加一个或者多个行，然后在行容器中插入列。

嵌套在列中的行也被划分为12等分。

<div class="row">
           <div class="col-md-2">
               <div class="row">
                   <div class="col-md-6">
                       嵌套元素
                   </div>
                   <div class="col-md-6">
                       嵌套元素
                   </div>
               </div>
           </div>
           <div class="col-md-2">
               原本的元素
           </div>
       </div>

三.常用样式

1. 排版：

Bootsrap和普通的html页面相同，定义标题使用标签<h1>到<h6>只不过bootstrap覆盖了默认样式，使用其在所有浏览器下显示效果一样，为了让非标题元素和标题元素使用相同样式，特意使用.h1 ~ .h6六个类名，同时后面可以及跟着一行小的副标题<small>或者使用.samll

(bootstrap标题加粗效果不明显，字体略微增大)

<h1>这是一个标题 <samll>这是一个副标题</samll></h1>
   <h2>这是一个标题</h2>
   <h3>这是一个标题</h3>
   <h4>这是一个标题</h4>
   <h5>这是一个标题</h5>
   <h6>这是一个标题</h6>
   <span class="h1">这是一个一级标题</span>

使用类名 .lead来突出强调文本的内容(作用为增大字号，加粗文本，并且对行高和margin也做了响应的处理)

2. 强调

定义了一套类名，称为强调类名，通过颜色来表示强调

  • .text-muted 提示，使用浅灰色
  • .text-primary 主要，使用蓝色
  • .text-success 成功，使用浅绿色
  • .text-info 通知，使用浅蓝色
  • .text-warning 警告，使用黄色
  • .text-danger 危险，使用褐色
<div class="text-muted">提示效果</div>
        <div class="text-primary">主要效果</div>
        <div class="text-success">成功效果</div>
        <div class="text-info">通知效果</div>
        <div class="text-warning">警告效果</div>
        <div class="text-danger">危险效果</div>
    </div>

3. 对齐效果

在css中使用text-align来实现文本对齐风格的设置

  • left 左对齐
  • right 右对齐
  • center 居中
  • justify 两端对齐

为了简化操作，Bootstrap通过定义四个类名来控制文本的对齐风格：

  • .text-left 左对齐
  • .text-right 右对齐
  • .text-center 居中对齐
  • .text-justify 两端对齐
<div class="container">
        <div class="text-left">左对齐</div>
        <div class="text-right">右对齐</div>
        <div class="text-center">居中</div>
        <div class="text-justify">两端对齐</div>
    </div>

4. 列表样式

html中列表分为三种

无序列表： 

<ul>
   <li></li>
</ul>
  • 有序列表
<ol>
   <li></li>
</ol>
  • 自定义列表
<dl>
       <dt>1</dt>
       <dd></dd>
       <dt>2</dt>
       <dd></dd>
</dl>

4.1 去点列表

使用class=“list-unstyle” 去除列表的装饰元素

<ul class="list-unstyled">
            <li>第一项</li>
            <li>第二项</li>
            <li>第三项</li>
</ul>

4.2 内联列表

class=”list-inline” ,把垂直列表换成水平列表，而且去掉项目符号，也可以说内联链表即使为了制作水平导航而生。

<ul class="list-inline">
            <li>列表项</li>
            <li>列表项</li>
            <li>列表项</li>
            <li>列表项</li>
            <li>列表项</li>
</ul>

4.3 定义列表

在原有的基础加入一些样式，使用类名: dl-horizontal 制作水平定义列表，当标题宽度超过160px时，将会显示三个省略号。

<dl class="dl-horizontal">
            <dt>自定义列表</dt>
            <dd>内容内容内容内容内容内容</dd>
            <dt>自定义列表自定义列表自定义列表自定义列表</dt>
            <dd>内容内容内容内容内容内容</dd>
 </dl>

4.4 代码风格

可以在网页上显示代码风格bootstrap提供了三种代码风格

  • <code></code> 用来显示单行代码样式。
  • <pre></pre> 用来显示多行块代码
    • 可以添加 class = “pre-scrollable” （height,max-height固定，为340px，如果超过此高度，显示滚动条）
  • <kbd></kbd> 用来显示用户输入代码样式(键盘的样式，用于显示快捷键)

5. 表格

bootstrap为表格提供了1种基础样式以及4种附加样式以及一个支持响应式的表格，只需要添加对应的类名就可以得到不同的表格风格

基础样式：

  1. .table 基础的表格样式
  2. .table-striped 斑马线表格
  3. .table-brodered 带边框的表格
  4. .table-hover 鼠标悬停高亮
  5. .table-condensed 紧凑形式表格，单元格没有内距或者内距小。

tr,td,th样式

类 描述 实例
.active 将悬停的颜色应用到行或者单元格
.success 表示成功的操作
.info 表示信息变化的操作
.warning 表示一个警告的操作
.danger 表示一个危险的操作

按钮

美化按钮

  • .btn 类名美化按钮

  • .btn-primary 蓝色

表单布局

基本表单结构时bootstrap自带的，个别的表单控件自动接受一些全局样式，使用基本表单需要遵循一些步骤：

  1. 向<form>标签中添加自定义属性 role=“form”
  2. 把标签和控件放到一个带有 .form-group的<div>中，这是获取最佳距离所必须的。
  3. 向所有文本元素<input> <textarea> <select> 添加 ,form-control 类名

水平表单

  • .form-horizontal

内联表单

  • .form-inline
