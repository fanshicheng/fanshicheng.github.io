above一.布局管理器

1.线性布局(LinearLayout)

线性布局采用嵌套垒放的形式进行位置的设置。

常用属性：

属性 值 作用
android:id @+id/id名 设置id标识
android:layout_width match_parent匹配父框架wrap_content匹配内容 宽度
android:layout_height match_parent匹配父框架wrap_content匹配内容 高度
android:background 背景
android:layout_padding dp 内边距
android:layout_margin dp 外边距
android:orinetation horizontal横向布局 vertical纵向布局 方向
android:gravity center left right 对齐方式
android:weight 权重
android:layout_width

用于设置宽度

  • match_parent 匹配父框架，得到父框架剩余所有宽度
  • wrap_content 匹配内容，设置正好包裹内容的宽度
  • xdp 自定义宽度
android:layout_height

用于设置高度

  • match_parent 匹配父框架，得到父框架剩余所有高度
  • wrap_content 匹配内容，设置正好包裹内容的高度
  • xdp 自定义高度
android:layout_padding

用于设置内边距，可以使用拆分属性单独设置

  • xdp

拆分属性：

  1. android:layout_paddingLeft
  2. android:layout_paddingRight
  3. android:layout_paddingTop
  4. android:layout_paddingButtom
android:layout_margin

用于设置外边距，可以使用拆分属性单独设置

  • xdp

拆分属性：

  1. android:layout_marginLeft
  2. android:layout_marginRight
  3. android:layout_marginTop
  4. android:layout_marginButtom
android:orinetation

用于设置排列方向

  • horizontal横向布局
  • vertical纵向布局
android:gravity

用于设置对齐方式

  • center 居中对齐s
  • bottom 底部对齐
  • center_horizontal 横向居中对齐
  • center_vertical 纵向居中对齐
  • left 左对齐
  • right 右对齐
android:weight

设置在布局中的权重，当宽度或高度设置为0dp时，将按照权重分配剩余空间。如果宽度或者高度不为零，则使用剩余空间出去设置的宽高再按照权重分配剩余的空间。

2.相对布局(RelativeLayout)

相对布局时相对于指定id属性的空间位置来设置位置。

属性 值 功能
android:layout_toLeftOf @id/id名 在指定id的左侧
android:layout_toRightOf @id/id名 在指定id的右侧
android:layout_alignBottom true 与指定id的下面对齐
android:layout_alignParentBottom true 与父框架底部对齐
android:layout_alignParentLeft true 与父框架左侧对齐
android:layout_alignParentRight true 与父框架右侧对齐
android:layout_layout_below @id/id名 在指定id的下面
android:layout_layout_below 在指定if的下方显示

二.控件

1. TextView

TextView控件用于显示文字和icm字符。

属性：

  • android:maxLines：”行数” 强制设置控件显示行数
  • android:textSize: “sp” 设置控件中的文本大小
  • android:textColor: “颜色” 设置控件中的文本颜色
  • android:ellipsize: “end” 设置控件文字超出范围替换为省略号显示
  • android:drawableBottom: “@drawable/路径” 设置文字 + icon
<TextView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="@string/textType"
    android:textColor="#ffff00"
    android:maxLines="1"
    android:ellipsize="end"
    android:drawableBottom=""
    android:drawableLeft=""
    android:drawableRight=""
    android:drawableTop="">
</TextView>

2. Button

Button继承自TextView类。

button可以通过android:background引入shape文件控制按钮的点击样式和圆角。

Button控件绑定点击事件：

也可以给其他控件设置点击事件。

button绑定点击事件步骤：

  1. 设置指定button控件的id属性。

    android:id="@+id/bt"

  2. 在相应视图的activity中定义对应的按钮对象。

    protected Button bt;

  3. 通过findViewById()方法获得按钮对象。

    bt = (Button) findViewById(R.id.bt);

  4. 为按钮对象添加点击事件监听器，重写click方法。

    bt.setOnClickListener(new View.OnClickListener() {});

public class MainActivity extends AppCompatActivity {
  	//在相应视图的activity中定义对应的按钮对象
    protected Button bt;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        setContentView(R.layout.activity_main);
      //通过findViewById()方法获得按钮对象。
        bt = (Button) findViewById(R.id.cql);
      
      //为按钮对象添加点击事件监听器，重写click方法。
        bt.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
              
            }
        });
    }
}

EditText

常用属性

  • android:hint=”文本框灰色提示” 设置文本框灰色提示
  • android:inputType=” “ 设置文本框显示形式
    • textPassword 密码
    • number 数字
    • date 日期
    • text 文本
  • android:drawableLeft=“路径” 将icon字符放置到EditText控件的左边
  • android:drawablePadding = “” 设置icon字符的内边距
<EditText
    android:id="@+id/edit2"
    android:layout_width="match_parent"
    android:layout_height="50dp"
    android:hint="提示信息"
    android:inputType="textPassword"
    android:paddingLeft="10dp"></EditText>

监听事件：

监听EditText输入框变化事件：

  1. 设置指定EditText控件的id属性。

    android:id="@+id/edit1"

  2. 在相应视图的activity中定义对应的EditText对象。

    protected EditText edit1;

  3. 通过findViewById()方法获得按钮对象。

    edit1 = (EditText) findViewById(R.id.edit1);

  4. 为EditText对象添加点击事件监听器，重写TextWatcher方法。

    et.addTextChangedListener(new TextWatcher() }

public class MainActivity extends AppCompatActivity {

     private EditText et;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        et = (EditText) findViewById(R.id.edit1);
        et.addTextChangedListener(new TextWatcher() {
            @Override
            //在文字改变之前
            public void beforeTextChanged(CharSequence charSequence, int i, int i1, int i2) {

            }

            @Override
            //在文字改变时
            public void onTextChanged(CharSequence charSequence, int i, int i1, int i2) {

            }

            @Override
            //在文字改变之后
            public void afterTextChanged(Editable editable) {

            }
        });

    }
}

RadioButton

单选按钮RadioButton和单选容器RadioGroup配合使用，在同一个单选容器中单选按钮只能被选中一个。

属性：

  • android:checked=”true” //默认选中 如果没有设置id此属性会失效
<RadioGroup
    android:id="@+id/rg"        
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:orientation="vertical" //设置单选按钮顺序>

    <RadioButton
        android:id="@+id/ma"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="男"
        android:checked="true"
        android:textSize="18sp"
        android:textColor="#ff6600">
    </RadioButton>

    <RadioButton
        android:layout_toRightOf="@id/ma"
        android:id="@+id/nv"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="女"
        android:textSize="18sp"
        android:textColor="#ff6600">
    </RadioButton>

</RadioGroup>

监听RadioGroup选择变化事件：

  1. 设置指定RadioGroup控件的id属性。

    android:id="@+id/rg"

  2. 在相应视图的activity中定义对应的RadioGroup对象。

    protected RadioGroup rg;

ImageView

图片视图

常用属性：

  • android:src=”@drawable/image” //设置图片内容
  • android:scaleType=”fitXY” //设置缩放模式
    • fitXY ：撑满控件，宽高比可能发生改变
    • fitCenter : 保持宽高比缩放，直至能够完全显示
    • centerCrop ： 保持宽高比缩放，直至完全覆盖控件，裁剪显示。
<ImageView
    android:layout_width="match_parent"
    android:layout_height="300dp"
    android:background="#ff9900"
    android:src="@drawable/image"
    android:scaleType="fitXY">
</ImageView>
