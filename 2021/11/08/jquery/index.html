jQuery

javascript库概念: 是一个封装好的特定的集合(方法和函数)，封装了很多预先定义好的函数。

常见的javascript库：

  • jQuery
  • Prototype
  • YUI
  • Dojo
  • Ext JS
  • 移动端的zepto

这些库都是对原生javaScript的封装，内部都是用javaScript实现的。

jQuery概念

jQuery是一个快速的，简介的javascript库，将js中的DOM操作做了封装，可以快速的查询使用里面的功能。

jQuery封装了javaScript常用的代码，优化了DOM操作，时间处理，动画设计和Ajax交互。

  • 轻量级，核心文件才几十kb，不会影响页面加载的速度。
  • 跨浏览器兼容，基本兼容了现在主流的浏览器。
  • 链式编程，隐式迭代。
  • 对事件，样式，动画支持，大大简化了DOM操作。
  • 支持插件扩展开发，有着丰富的第三方的插件：树形菜单，日期控件，轮播图。
  • 免费开源。

jQuery入口函数

可以设置入口函数，将script代码写在入口函数中，等待页面加载成功后再执行。

方式一：

$(document).ready(function(){
        //函数代码
    })

方式二：

$(function(){
         //函数代码
    })
  1. 等待dom结构渲染完毕即可以执行内部代码，不必等到所有外部资源加载完成。
  2. 相当于原生js中的DOMContentLoaded。
  3. 不同于原生js的load事件是等待页面文档，外部js文件，css文件，图片加载完成才执行内部代码。

jQuery顶级对象

$ 是jQuery的别称，在代码中可以使用jQuery代替 $ .

$ 是jQuery的顶级对象，相当于原生的javaScript中的window，把元素利用 $ 包装成jQuery对象，就可以调用jQuery的方法。

dom对象和jQuery对象

  1. dom对象：使用原生js获取过来的对象。
  2. jQuery对象：使用jQuery方式获取过来的对象就是jQuery对象，本质就是使用$把dom对象进行了包装。(是以数组的形式包装的)
  3. jQuery对象只能使用jQuery的方法，dom对象使用原生js的方法和属性。

jQuery和DOM对象的相互转换

因为原生js比jQuery更大，原生的一些属性和方法jQuery没有完全实现，要想使用这些属性和方法，需要把jQuery对象转换为DOM对象才能使用。

  1. DOM对象转换为jQuery对象： $(DOM对象)

    $('div');

  2. jQuery对象转换为DOM对象: jQuery对象[索引] || jQuery对象.get(索引)

    $('div')[0]

jQuery常用API

  • $(“选择器”).css() //设置css属性
  • $(“选择器”).事件名(function(){}) //设置事件监听
  • $(“选择器”).show() //显示元素
  • $(“选择器”).hide() //隐藏元素
  • $(this) //获得当前元素
  • $(this).index() //获得当前元素的索引
  • $(“选择器”).eq(索引) //选择指定索引的元素
  • $(“选择器”).hover(进入函数，离开函数); //鼠标进入和离开触发相应的函数
  • $(“选择器”).hover(函数) //鼠标进入和离开都触发一次函数
  • $(“选择器”).change(函数) //当指定组件发生变化之后会执行指定的函数
  • toFixed(保留位数) //设置指定数值的小数点后位数

jQuery 基础选择器

原生JS获取方式很多，很杂且兼容性情况不一致，jQuery将选择器做了封装，使得获取元素统一标准。

jQuery选择器拿取的元素以数组的方式放置到对象数组中。

语法：

jQuery样式操作

操作css方法

jQuery 可以采用css方法来修改来修改元素样式，也可以操作类，修改多个样式。

1.参数只写属性名，则是返回属性值

$(this).css("color");   //获取当前元素的color属性

2.参数是属性名，属性值，使用逗号分割，是设置一组样式，属性必须加引号，属性值如果是数字可以不用加单位和引号。(可以使用变量名)

$(this).css("color","red");   //设置当前元素的color属性
$(this).css("height",100);
  1. 参数使用对象的形式，可以方便的设置多组样式，属性名和属性值使用冒号隔开，属性可以不用加引号。
    • 如果是符合属性必须使用驼峰命名法
$(this).css(
  		height:100,
         width:400，
  		backgroundColor:"red";
           );
设置类样式方法

作用等同于之前classList，可以操作类样式，注意操作类里面的参数不能加点。

原生js中的className会覆盖原先里面的类名。

jQuery里面类操作只是对指定类进行操作，不影响原先的类名。

  • 添加类 $(选择器).addClass(“类名”)

    $("div").addClass("className")

  • 删除类 $(选择器).removeClass(“类名”)

    $("div").removeClass("className")

  • 切换类 $(选择器).removeClass(“类名”)

    • 如果有指定类名则删除，如果没有就添加
    $("div").toggleClass("className")

jQuery元素操作

元素操作主要是 遍历，创建，添加，删除元素操作。

jQuery 隐式迭代是对同一类元素做了同样的操作。如果想要给同一类元素做不同的操作，就需要使用遍历。

遍历：

语法：

$(“div”).each( function( index,domEle ){ } );

  1. each()方法遍历匹配的每一个元素，主要是用DOM处理。
  2. 里面的回调函数有2个参数：index是每个元素的索引号，demEle是每个DOM元素对象==(不是jQuery对象)==。
  3. 使用jQuery方法，需要将dom对象转换为jQuery对象。 $(dom对象)
$("div").each(function(index,domEle){
  	//index 获得当前索引号
  	//domELe  获得当前dom元素  $(domEle)
  
})

语法：

$.each(object,function ( index,element ) );

  1. $.each()方法可以用于遍历任何对象，主要用于数据处理，比如数组，对象。
  2. 函数中有两个参数，index是每个元素的索引号，element是遍历内容。
//遍历元素对象
$.each($("div"),function(index,ele){
  	//index 获得当前索引号
  	//ele  获得当前dom元素  $(domEle)
})
//遍历数组
var arr = [0,1,2,3];
$.each(arr,function(index,ele){
  	//index 获得当前索引号
  	//ele  获得当前dom元素  $(domEle)
})
//遍历对象
$.each({name:"doubao",age:18},function(i,ele){

  	//ele  获得当前属性值
