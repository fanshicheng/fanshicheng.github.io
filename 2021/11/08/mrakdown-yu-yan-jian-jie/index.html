mrakdown语言简介

  • Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

  • Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。

  • Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

  • Markdown 编写的文档后缀为 .md, .markdown。

mrakdown语言主要格式总结

一.标题

1.一级标题：

“#” + 标题题目 快捷键：选中标题内容 ctrl + 1

# 一级标题

2.二级标题：

“##” + 标题题目 快捷键：选中标题内容 ctrl + 2

## 二级标题

3.三级标题：

“###” + 标题题目 快捷键：选中标题内容 ctrl + 3

### 三级标题

4.四级标题：

“####” + 标题题目 快捷键：选中标题内容 ctrl + 4

#### 四级标题

5.五级标题：

“#####” + 标题题目 快捷键：选中标题内容 ctrl + 5

##### 五级标题

6.六级标题：

“######” + 标题题目 快捷键：选中标题内容 ctrl + 6

###### 六级标题

二.引用文字

1.引用

“>” + 要引用的内容

> + 空格 + 要引用内容

*举例：

引用内容

2.多级引用

“>” + 空格 + “>” + “空格” + 引用内容

> > 内容
> > > 内容
> > > > 内容

*举例：

引用

引用

三.列表

1.有序列表

数字键 + “.” + 空格 + 内容

1. 内容

*举例：

  1. 内容

  2. 内容

2.无序列表

“*” + 空格 + 内容

* 内容

*举例：

  • 内容
  • 内容

四.代码块

“```” + 语言名字

​``` java\c\其他

*举例：

java
javascriot

五.表格

“| 内容 | 内容| 内容|”

| 表头 | 表头 | 表头|

*举例：
| 姓名 | 年龄 | 代号 |
| —- | —- | —- |
| | | |

六. 图片

“!” + “[ 显示内容 ]” + “( 图片地址)” //快捷键：Ctrl + Shift + I

![图片名](地址)

*举例：

##七.超链接

“[显示文字]” + “(超链接地址)” //快捷键：ctrl + k

[超链接](https://wallhaven.cc/)

*举例：这是一个超链接

##八.斜体

“* “ + 文字 + “* “ “ _ “ + 文字 + “ _ “ //快捷键：ctrl + i

*文字*    _文字_

*举例：斜体文字：文字 原版文字：文字

##九.加粗

“ ** “ + 文字 + “ ** “ “ __” + 文字 + “ __” //快捷键：ctrl + b

**文字**        __文字__

*举例：加粗文字： 文字 原版文字：文字

十.加粗斜体

“*** “ + 文字 + “*** “

***文字***

*举例：加粗斜体文字: 文字 原版: 文字

十一.删除线

“~~ “ + 文字 + “~~ “ //快捷键：alt + shift + 5

~~文字~~

*举例：文字

十二.下划线

< u >下划线 //快捷键：ctrl + u

<u>文字</u>

*举例： 文字

十三.表情

“:” + 表情代码

:100

*举例：:happy:

十四.上标/下标

1.上标：

文字 +” ^ “ +上标 + “ ^ “ “ “ + 上标 + “< /sup>” 注：此功能需要在偏好设置中打开

^上标^                     <sup>上标</sup>

*举例：H^2^

2.下标：

文字 + ““ + 下标 + ““ ““ + 下标 + “</ sub>” 注：此功能需要在偏好设置中打开

~下标~                      <sub>下标<\sub>

*举例：H2O

十五.高亮

“= =” + 高亮文字 + “= =” 注：此功能需要在偏好设置中打开

==高亮文字==

*举例： ==高亮==

十六.代码标注

“ ` “ + 需要标注的代码或文字 + “ · “

` 需要标注的代码或者文字 `

*举例：printf("");

十七.文本居中

< center > + 居中的内容 + < /center>

<center>居中的内容</center>

*举例：

居中的内容
---

十八.转义字符

\ 反斜线

` 反引号

星号 *

_下划线

{} 花括号

[] 方括号

() 小括号

# 字号

+加号

-减号

. 英文句点

! 感叹号

以上字符可以通过在字符前添加 \ 反斜线来转义。
