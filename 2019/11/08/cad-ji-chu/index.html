一.cad基础部分命令

使用cad2021总结

1. 直线

控制台快捷键： L

默认画完一条直线不会取消，而是可以继续点击下一段直线的终点。

多段直线或者多段直线和多端圆弧可以组合为多段线。

多段线与直线的区别是多段线可以设置宽度。

可以使用F8来打开和关闭正交模式，来达到自由画直线或者只画平行线或者垂直线。

操作过程：

  1. 单击直线工具(或者快捷键：L + 空格) - 》
  2. 在图纸上单击选择直线的起点 -》
  3. 在图纸上单击选择直线的终点。 （或者确定起点后输入直线的长度 + 空格）

中间的参数：

无

2. 圆

控制台快捷键： C

画圆有多种方式： (半径和直径可以输入)

  1. 圆心 + 半径()
  2. 圆心 + 直径
  3. 确定的圆上的两点
  4. 确定的圆上的三点
  5. 确定两条直线的相切
  6. 确定三条直线的相切

操作过程：

  1. 单击圆工具(或者快捷键：C + 空格) - 》 /* 默认的画圆方式可能和要用的不相同，需要手动下拉选择
  2. 选择画圆必要的参数（或者输入相应的参数来选择要使用的画圆方式）

中间的参数：

3P：三点画圆

2P：两点画圆

T: 切点+切点+半径画圆(半径可以输入)

补充:

新版可以直接在画圆工具中选择三切点,

如果是老版,三切点比较繁琐:

1.选择画圆工具(或者快捷键 C + 空格)

2.输入参数 3P

3.输入参数 tan 单击要切的一条线 *3

3. 圆弧

控制台快捷键： ARC

画圆弧有多种方式： (都需要三个条件)

  1. 起点 + 圆心 + 端点
  2. 起点 + 圆心 + 角度
  3. 起点 + 圆心 + 长度
  4. 起点 + 端点 + 角度
  5. 起点 + 端点 + 方向
  6. 起点 + 端点 + 半径
  7. 圆心 + 起点 + 端点
  8. 圆心 + 起点 + 角度
  9. 圆心 + 起点 + 长度

操作过程：

  1. 单击圆弧工具(或者快捷键：ARC + 空格) - 》 /* 默认的画圆弧度方式可能和要用的不相同，需要手动下拉选择
  2. 选择画圆弧必要的参数（或者输入相应的参数来选择要使用的画圆弧方式）

中间的参数：

C : 选择圆心

补充:

4. 多段线

控制台快捷键： PL

默认画完一段多段线不会取消，而是可以继续点击下一段多段线的终点,可以绘制圆弧。

多段直线或者多段直线和多端圆弧可以组合为多段线。

多段线与直线的区别是多段线可以设置宽度。

可以使用F8来打开和关闭正交模式，来达到自由画直线或者只画平行线或者垂直线。

操作过程：

  1. 单击多段线工具(或者快捷键：PL + 空格) - 》

  2. 在图纸上单击选择多段线的起点 -》

    2.5 绘制起点后可以通过输入参数来控制多段线的样式及宽度.

  3. 在图纸上单击选择多段线的终点。（或者确定起点后输入多段线的长度 + 空格）

单击选择起点后的参数:

A: 转换多段线为圆弧样式.

H(半宽): 设置多段线宽度,输入H 后第一个输入 参数为多段线起点宽度,第二个参数为多段线终点宽度.

W: 设置多段线宽度,输入H 后第一个输入 参数为多段线起点宽度,第二个参数为多段线终点宽度.

D: 在圆弧样式下,输入D来控制圆弧的弯曲方向,输入D后,通过选定方向线的朝向来控制圆弧朝向.

补充:

直线转换多段线:

控制台输入参数 : PE

控制台输入参数: M (控制多条多段线)

选中多段直线 +　空格

控制台输入参数: Y (确定组成多段线)

控制台输入参数: J(或者快捷菜单选择合并) + 空格

5. 矩形

控制台快捷键： RECTANG

操作过程：

  1. 单击矩形工具(或者快捷键：L + 空格) - 》
  2. 在图纸上单击选择矩形的起点(或者输入参数控制矩形的宽度和厚度,圆角,倒角,标高) -》
  3. 在图纸上单击选择矩形的终点。

未单击选择起点的参数:

C:设置矩形的倒角,输入参数C+空格后,第一个参数为第一个倒角,第二个参数为第二个倒角.

E:标高

F:设置矩形的圆角,输入参数F +空格后,再输入参数为圆角的半径.

T:设置矩形的厚度.(在3d视图明显)

W:设置矩形边的宽度.

单机选择起点后的参数:

A:面积

D:设置矩形的尺寸,输入参数D + 空格后再输入矩形的长度和宽度来构造矩形.

R:旋转矩形.

6. 多边形

控制台快捷键： POL

操作过程：

  1. 单击多边形工具(或者快捷键：POL + 空格) - 》
  2. 在控制台输入侧边数量 + 空格 -》
  3. 在图纸上单击选择指定多边形的中心 3.或者在控制台输入参数E,来控制边的长度
  4. 在控制台输入参数I或者C来选择内接还是外切于圆．　４.在图纸上单机指定第一个端点.
  5. 拉伸或者输入半径构造多边形. 5. 点击选择有这条边的终点或者输入边的长度构造多边形.

未单击选择起点的参数:

E:转换到通过边来构造多边形.

单机选择起点后的参数:

I : 构造内接于圆的多边形.

C: 构造外切于圆的多边形.

7. 移动和复制

控制台快捷键：

复制: copy

移动操作过程：

  1. 单击移动工具
  2. 选择要移动的对象 + 空格
  3. 点击选择要移动对象的基点(输入参数D,用输入坐标的方式移动对象)
  4. 单击选择移动的位置

复制操作过程：

  1. 单击复制工具(快捷键copy + 空格)
  2. 选择要复制的对象 + 空格
  3. 点击选择要复制对象的基点(输入参数D,用输入坐标的方式移动对象)
  4. 单击选择复制的位置

单机选择起点后的参数:

复制:

M: 选择复制的模式,是只黏贴一次还是允许继续黏贴.

D:通过坐标点选择黏贴的位置.

A:阵列.可以使被黏贴对象以指定数量阵列排序黏贴.

8. 拉伸

控制台快捷键：STR

操作过程：

  1. 单击拉伸工具
  2. 在图纸上从右下到左上选择要拉伸的对象 + 空格 (要是把对象全选会变成移动)
  3. 单击选择拉伸对象的基点,拉伸对象.

单机选择起点后的参数:

D:转换为位移.

补充:

选择对象时一定要从右下往左上选择,否则会选择不中.

9. 旋转

控制台快捷键：RO

操作过程：

  1. 单击旋转工具(快捷键 RO + 空格)
  2. 选择要旋转的对象 + 空格
  3. 指定旋转对象的旋转基点.
  4. 旋转到指定方向.(控制台输入逆时针旋转度数)

单机选择基点后的参数:

C: 会复制原来的对象进行旋转

R: 参照对象的两个基点形成的直线来于选择要旋转的基点来进行旋转 .

10. 镜像,缩放,对齐缩放

控制台快捷键：

镜像: MI

缩放: SC

对齐缩放: AL

镜像操作过程：

  1. 单击镜像工具(快捷键 MI+ 空格)
  2. 选择要使用镜像的对象 + 空格
  3. 选择指定镜像线 + 空格
  4. 输入参数 Y或者N 来选择是否保留之前的对象.

缩放操作过程

  1. 单击缩放工具(快捷键 SC + 空格)
  2. 选择要使用缩放的对象 + 空格
  3. 指定要缩放的基点 + 空格
  4. 移动缩放(或者输入缩放的倍数)

缩放参数

C: 复制选定对象再进行缩放,不影响原来的对象.

R: 指定两个点组成的一条线为参照先进行缩放.

对齐缩放操作过程

  1. 在控制台输入Al + 空格
  2. 选择要对齐缩放的对象
  3. 分别选择源点和对应的目标点,
  4. 输入参数 Y或N来控制对齐缩放

11. 延伸和修剪

控制台快捷键：

修剪 : Trim

延伸 : EXTEND

修剪操作过程：

  1. 选择修剪工具.
  2. 选择要修剪的对象.

修剪参数

T: 修剪边,确定修剪的范围,使得修剪可以无视交叉的线段

C: 窗交,使得修剪不能无视交叉的线段.

O: 模式

p: 投影

延伸操作过程

  1. 选择延伸工具.
  2. 选择要延伸的对象.

延伸参数

B:边界,选择延伸到的边界,使得延伸不会被非边界所阻挡.

p:投影.

o: 模式.

12. 圆角和倒角

控制台快捷键：

圆角操作过程：

  1. 选择圆角工具.
  2. 控制台输入参数 R + 空格(r为圆角所在圆的半径)
  3. 选择圆角的两边. (两边可以使连接的也可以使不相交的两条线)

圆角参数

R: 选择半径创建圆角,输入指定半径创建圆角.

倒角操作过程

  1. 选择倒角工具.
  2. 控制台输入参数 D + 空格.
  3. 输入第一个倒角距离 + 空格
  4. 输入第二个倒角距离 + 空格
  5. 分别选择倒角边 (倒角边选择顺序和输入倒角顺序相同) (两边可以使连接的也可以使不相交的两条线)

倒角参数

B:边界,选择延伸到的边界,使得延伸不会被非边界所阻挡.

D:输入倒角距离来生成倒角

补充:

可以使用圆角和倒角工具使两条不相交的直线相交.

13. 曲线和曲线相交

控制台快捷键：

曲线生成方式有两种: 曲线拟合 和 控制点 两者可以互相转换(生成去先后可以选中曲线,曲线头部有个小三角.点击后可以选择控制曲线的方式.)

操作过程：

  1. 单击曲线生成工具
  2. 在图纸上单击进行点位的选择.

连接两条曲线操作过程:

  1. 选中光顺曲线共工具
  2. 分别选中需要连接的两条曲线.

参数:

14. 阵列

控制台快捷键：

阵列分为 : 矩阵阵列,环形阵列,路径阵列

矩形阵列 操作过程：

  1. 选择相应的矩形阵列工具
  2. 选中需要使用阵列的对象 + 空格
  3. 可以在上方菜单处阵列创建处来控制阵列的间距和数量.

路径阵列操作方式:

  1. 选择相应的矩形阵列工具.
  2. 选中需要使用阵列的对象 + 空格 .
  3. 选中要排列的路径.
  4. 在菜单的阵列创建中选择基点.
  5. 选择对应路径的对象基点.

环形阵列 操作过程：

  1. 选择相应的环形阵列工具
  2. 选中需要使用阵列的对象 + 空格
  3. 可以在上方菜单处阵列创建处来控制阵列的间距和数量.

15. 参数化

在上方处选择参数化菜单

操作过程：

  1. 选择参数化菜单.
  2. 选择相应的选项.
  3. 分别选择相应选项的对象. (以第一个选择的对象为基准,移动或者变换以后选择的对象)

参数化选项

  1. 重合:
  2. 共线
  3. 同心
  4. 平行
  5. 垂直
  6. 水平
  7. 竖直
  8. 相切
  9. 平滑
  10. 相等
  11. 对称
